# Техническое задание: Система генерации ТЗ из голосовой речи
# Важно по стекам использовать бесплатные сервисы для трансрибации тестка , что касаемо генерации ТЗ из полученого текста я еще думаю через какое AI делать (можно предложить вариаты пока идет разработка  и тестирование) 

## 1. Цель проекта
Создать веб-приложение для формирования технических заданий на основе голосовой речи с последующей структуризацией, редактированием, добавлением вложений и экспортом в PDF.

## 2. Основные возможности
- Голосовой ввод речи в реальном времени
- Преобразование речи в текст
- Автоматическая структуризация текста в пункты ТЗ с помощью AI
- Редактирование пунктов вручную
- Добавление скриншотов и файлов к каждому пункту
- Указание оценки времени по каждому пункту
- Экспорт готового ТЗ в PDF
- Возможность делиться ТЗ внутри компании
- Интеграция с Bitrix24 (создание задач на основе ТЗ)

## 3. Роли пользователей
- **Администратор**: управление системой, пользователями и интеграциями
- **Менеджер**: создание и редактирование ТЗ
- **Исполнитель**: просмотр ТЗ

## 4. Модуль авторизации
- Регистрация по email и паролю
- Вход в систему
- Восстановление пароля
- Разграничение прав доступа

## 5. Модуль создания ТЗ
- Создание проекта/клиента
- Создание ТЗ внутри проекта
- Разделы и пункты
- Drag & Drop сортировка
- Оценка времени по каждому пункту

## 6. Голосовой ввод
- Запись через микрофон
- Отображение текста в реальном времени
- Поддержка русского языка

## 7. AI-структурирование
- Преобразование текста в список задач
- Формирование структуры документа
- Уточняющие вопросы

## 8. Вложения
- Добавление изображений
- Добавление файлов
- Привязка к пунктам

## 9. Экспорт
- Генерация HTML-версии
- Экспорт в PDF

## 10. Интеграция Bitrix24
- Создание задач
- Назначение исполнителей
- Передача описания и пунктов
